InfoPC — точное сканирование вашего ПК

InfoPC собирает основные характеристики: процессор, видеокарта, оперативная память по модулям, диски и материнская плата. Интерфейс без шума — стекло, мягкий свет, чёткая типографика. Есть сравнение вашего ПК с минимальными и рекомендуемыми требованиями игр.

Важно: антивирус или Windows SmartScreen могут предупредить при установке — файлы сейчас без цифровой подписи. Приложение безопасное. Если увидите SmartScreen, нажмите «Подробнее» → «Выполнить в любом случае».

Ключевые особенности
Скан железа
CPU, GPU, RAM (модули), материнская плата и диски — аккуратно и без лишних экранов.
Список игр
Сравнение MIN/REC требований популярных игр с вашим ПК — наглядно и просто.
Стеклянный UI
Типографика, мягкий свет, плавные переходы — интерфейс выглядит и ощущается живым.
Системные требования
